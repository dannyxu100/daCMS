关键字百度排名批量查询工具使用说明：

1，本工具使用PHP5开发，程序功能很简单，没有特殊要求，PHP4应该也可以。
2，用于批量查询网站关键字在百度的排名，例如查询aaa.com网站的 关键字1、关键字2、……、关键字n在百度的排名位置分别是多少。
3，程序原理很单一，就是采集百度的查询结果页面。
4，运行环境，apache+php，建议在您电脑上搭建一个apache+php环境，例如wamp整合包等等、网上很多，许多开源的cms也提供整合包。
5，下载baiek.rar文件，其中包括
	--index.php（可以改名）
	--Snoopy.class.php  （很好用的PHP采集类）
	--readme.txt （就是本文）
	--kw.xsd （xml schema）
	--kw.xml （关键词列表文件，要查询的域名与其关键字名单，一条一条写进去就可以了，可以用记事本编辑）
	--kw1.xml kw2.xml kw3.xml （如果有必要，这些是xml范例文件）
6，说说怎样使用吧，
	1）首先把下载到的rar文件解压到你的主机空间或虚拟主机(apache+php)，或者解压到本级搭建的环境下，目录随意
	2）直接打开浏览器，正确路径下运行index.php文件即可，
	3）运行界面会提示用户输入关键字清单文件，你需要参考kw.xml文件的格式做一份自己的关键字清单文件，例如abc.xml，文件中添加好自己要查询的域名与关键字以后，输入框填写abc.xml，然后点击查询按钮，等着看结果吧。

7，编辑关键字清单文件(.xml)需要注意的
	1）范例的文件的格式不要改变，就在原基础上增减。
	2）xml文件中，<kgroup>...</kgroup>，这就是一组，
	子项目包括了1个所要查询的网站域名<kdomain>你的域名</kdomain>，
	子项目还包括多个关键字<kword>你的关键字</kword>。
	例如网站域名是abc.com，我想查询 关键字a、关键字b、关键字c的百度搜索结果中，这个网站的排名情况，那么<kgroup>组就应当编辑成这个样子
	<kgroup>
		<kdomain>abc.com</kdomain>
		<kword>关键字a</kword>
		<kword>关键字b</kword>
		<kword>关键字c</kword>
	</kgroup>

	我们可以建立多个<kgroup>组，也就可以批量查询多个网站、多个关键字在百度的排名。

8，程序执行的过程就是依次读取各个关键字的百度搜索结果页面，检查我们域名的网页是否出现在搜索结果中、排名如何，实际就是机器查询。

9，一个xml关键字清单文件中放置几百个关键字问题不大，但是百度对来自同一IP的频繁查询会有所抵制，所以数量不要太大，可以多做几个。
